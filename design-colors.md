# DEN BMX Design Colors

Complete color palette for the DEN BMX website design system.

## Primary Colors

### White
- **Hex:** `#FFFFFF`
- **RGB:** `rgb(255, 255, 255)`
- **HSL:** `hsl(0, 0%, 100%)`
- **OKLCH:** `oklch(1 0 0)`
- **Usage:** Lettering and text

### Black
- **Hex:** `#000000`
- **RGB:** `rgb(0, 0, 0)`
- **HSL:** `hsl(0, 0%, 0%)`
- **OKLCH:** `oklch(0 0 0)`
- **Usage:** Primary canvas/background

## Accent Colors

### Primary Accent (Neon Green)
- **Hex:** `#00FF0D`
- **RGB:** `rgb(0, 255, 13)`
- **HSL:** `hsl(123, 100%, 50%)`
- **OKLCH:** `oklch(0.87 0.29 143)`
- **Usage:** Main brand color, primary UI elements, borders, highlights

### Secondary Accent (Blue)
- **Hex:** `#0073FF`
- **RGB:** `rgb(0, 115, 255)`
- **HSL:** `hsl(213, 100%, 50%)`
- **OKLCH:** `oklch(0.59 0.23 259)`
- **Usage:** Secondary UI elements, alternative accents

### Alternate Primary Accent (Magenta/Pink)
- **Hex:** `#FF00F2`
- **RGB:** `rgb(255, 0, 242)`
- **HSL:** `hsl(303, 100%, 50%)`
- **OKLCH:** `oklch(0.69 0.31 331)`
- **Usage:** Special highlights, animations, alternate branding

### Alternate Secondary Accent (Orange)
- **Hex:** `#FF8C00`
- **RGB:** `rgb(255, 140, 0)`
- **HSL:** `hsl(33, 100%, 50%)`
- **OKLCH:** `oklch(0.75 0.18 58)`
- **Usage:** Special highlights, call-to-action variations

## Color Usage Guidelines

- **Background:** Black (`#000000`)
- **Text:** White (`#FFFFFF`)
- **Primary Accent:** Neon Green (`#00FF0D`) - Used for borders, buttons, highlights, and key UI elements
- **Secondary Colors:** Use blue, magenta, or orange for variety and special emphasis

## Design Tool Import

For Figma, Sketch, or Adobe XD:
1. Use the JSON file (`design-colors.json`) with design token plugins
2. Or manually import using the hex codes above
3. For CSS variables, see `src/app/globals.css`

## Notes

- All colors are in sRGB color space
- The primary accent green (`#00FF0D`) is the main brand color and should be used consistently
- The alternate colors provide flexibility for special features and animations
